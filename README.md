## Description of Scripts
1. **System Identity:** Displays kernel version, user info, and uptime.
2. **Package Inspector:** Checks if software is installed and prints its philosophy.
3. **Disk Auditor:** Loops through system directories to check size and permissions.
4. **Log Analyzer:** Reads log files line-by-line to count specific keywords.
5. **Manifesto Generator:** An interactive script that creates a FOSS philosophy file.

## Instructions to Run
1. Open your Linux terminal.
2. Clone this repo: `git clone [Your Repo URL]`
3. Navigate to the folder: `cd oss-audit-[rollnumber]`
4. Give permission to the scripts: `chmod +x *.sh`
5. Run a script: `./identity.sh`

## Dependencies
* Bash Shell (Standard on Linux)
* Coreutils (grep, awk, du, ls)
